# Aplikasi Input Nilai Mahasiswa 
Code Lab RPL - Pertemuan 9 (Implementation & Testing Phase)  
Dibuat oleh: **Jihan Nabillah**

## Teknologi
- HTML, CSS, JavaScript  
- Firebase Firestore  
- Bootstrap  
- GitHub Pages (Deployment)

## Fitur
- Form input nilai mahasiswa
- Validasi input (wajib diisi, nilai 0–100)
- Simpan ke Firestore
- Tampilkan data dalam tabel
- Desain UI/UX modern, bersih, responsif

## Deployment
Aktifkan GitHub Pages melalui tab *Settings → Pages* untuk mempublikasikan aplikasi online.
